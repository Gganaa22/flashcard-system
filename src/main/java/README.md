# Flashcard System

## Ажиллуулах
flashcard <cards-file> [options]

## Сонголтууд
- `--order random | worst-first | recent-mistakes-first`
- `--repetitions <num>`
- `--invertCards`
- `--help`

## Cards файлын формат
Асуулт|Хариулт

### Жишээ
Capital of France|Paris
2+2|4
Capital of Japan|Tokyo

## Амжилтууд
- **CORRECT** — Сүүлийн тойрогт бүгдийг зөв хариулсан
- **REPEAT** — Нэг картад 5-аас олон удаа хариулсан
- **CONFIDENT** — Нэг картад 3-аас дээш удаа зөв хариулсан
- **SPEEDRUN** — Дундаж 5 секундээс доош хугацаанд хариулсан
